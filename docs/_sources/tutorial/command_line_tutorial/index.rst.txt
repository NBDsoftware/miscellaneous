From Command Line
==========================

Search all molecules present in all sdf files
-----------------------------------------------------

It will output a sdf2_output.sdf and sdf1_output.sdf with the common molecuels
and the different conformations of each of one of them in each file.

    ::

     python -m miscellaneous.search_molec_sdfs <list of sdf files>

     python -m miscellaneous.search_molec_sdfs sdf1.sdf sdf2.sdf
